Summary

-This is a smart todo app that allows user to keep track of his daily tasks.
-It allows users to automatically generate task description for the given task through AI model gpt2.
-User can set custom reminders of their choice but by default the remider has been set for 5 seconds when a new task is added.


Installation

-Clone the repository frm github
-Install the dependencies by running the command "pip install -r requirements.txt"
-Run the app by giving the command "flask run"  in the terminal


Features

-Add task.
-Update the status of a task.
-Delete a task.
-Custom Reminders.
-Default Reminders.
-Update the task status through reminders.
-AI generated description.
-Informative Alert messages according to situation.
